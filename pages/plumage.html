Plumage

HTML • CSS • JS • UI • UX

About

Plumage is a front end framework that was created as part of a final project in a class. The three requirements were that it had to be responsive, have css componenets and javascript plugins. Plumage was created in a one month time period.

Challenge

I took the lead role in my team and it was a new experience for me. Besides having to work on the project myself I was now in charge of dividing the work in a fair manner. I was also in charge of creating a responsive grid system that worked for all display sizes.

Solution

This is one of my favorite projects due to how much I learnt as we worked. This is where everything came together. The git workflow is someting that I have come to appreciate as it makes working as a team easier. Learning how to compromise with team mates is a skill that I have found to be important. It is important to listen to others ideas as the best ones come from working together.
