Todo List

About

A simple todo list web app. This was built as a personal project of mines to help me learn javascript. Design was taken from designer and devloper Max Sandelin. This app was built mobile first with a simple responsive design for desktops. This is a project that I wish to further improve on and add more features with time.

Challenge

Javascript is one of the more challenging things I have enjoyed learning and improving on. It is something that humbles me and there is always something that I will not know. This app at times became frustrating when I would not know how to make something work.

Solution

Despite feeling like I knew nothing, I did not let that demotivate me. I kept working and hacking away. Learning the parts of javascript I did not know yet was important as then I would know what I needed to research and read. Eventually with time the app came together and is functioning.
